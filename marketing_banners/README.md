- **Instagram**: Use Square (1:1) or Portrait (4:5) images.
- **Stories**: Use the Vertical (9:16) Green Story image.
- **Facebook**: Use Landscape images (Blue, Dark Blue, Orange) or Square.
- **WhatsApp**: The Square images are perfect for sending via WhatsApp.
